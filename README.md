# Valparaiso University's ACM Chapter Website

A website for the Valparaiso University Chapter of the Association for Computing Machinery.

## Administration Guide

Consult `documentation/pdf/documentation.pdf` for any administrative details regarding modifications to the website.

## Website Features
- Responsive design for different device types (mobile, tablet, laptop, desktop, etc.)
- A navigation bar for seamless website navigation
- Buttons for accessing Valpo ACM's social media, platforms, and communication methods
- A brand new landing page
- A reworked way to display E-Board Members
- A card to display events, including event details, an event description, a resources folder after an event concludes, and an option to add an event to Google Calendar
- A project card to show off projects created by the chapter 
- A button to display the Valpo ACM Constitution on a new page

## The Live Website
To view the live website, please go to: https://valpo-acm.github.io/
